What are Skill Templates?

Skill templates are pre-built code templates for creating custom skills for Amazon Alexa. Every skill tempate include all of the code you'll need along with a tutorial video that will step you through understanding, modifying, and deploying the template.

How are Skill Templates different from the Skill Blueprints provided by Amazon?

Blueprints are a great way to build personal skills but they're limited when it comes to building more complex skills. Skill Templates are designed to provide more flexibilty while minimizing the learning curve required for building custom skills.
