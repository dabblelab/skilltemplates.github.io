Getting Started - Amazon Alexa

The skill templates for Amazon Alexa are designed to be used with the Alexa Skills Kit Command-Line Interface. So to use them, you'll need the following.

  • An Amazon developer account. Register at developer.amazon.com
  • An AWS account. Signup at aws.amazon.com
  • Node.JS installed. Follow the steps in these videos to install on Mac, Windows, or Linux.
  • The Alexa Skills Kit CLI. Alexa Skills Kit Command-Line Interface
