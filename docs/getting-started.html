Getting Started - Amazon Alexa

The templates on SkillTemplates.com are all designed to be used with the Alexa Skills Kit Command-Line Interface (aka, the ASK CLI). The following video will help you get the ASK CLI installed and configured. After the ASK CLI is all setup, you'll have everything you need to start building custom Alexa skills using skill templates.

Setting up the Alexa Skills Kit CLI

{% include video-youtube.html video_id="lcJ_K7dTjW0" %}
  • Install Node.js v8.10 from nodejs.org
  • Install git from git-scm.com
  • Get an Amazon Developer Account at developer.amazon.com
  • Get an AWS account. Signup at aws.amazon.com
  • Reference the ASK CLI Quick Start Guide for details.
