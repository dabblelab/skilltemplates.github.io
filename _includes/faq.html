What are skill templates?
Skill templates are pre-built code templates for creating custom skills for Amazon Alexa. Every skill tempate include all of the code you'll need along with a tutorial video that will step you through understanding, modifying, and deploying the template.
How do I use skill templates?
For details on using skill templates please see our Getting Started Guide.
Where are the templates for Google Assistant and Microsoft Cortana?
Stay tuned - templates for Google Assistant and Microsoft Cortana are coming soon.
Can I contribute templates?
Absolutely! Just send an email to skilltemplates@dabblelab.com with a link to a public git repo containing the code you'd like to share as a template. We'll follow-up with you from there.
How do I get support for a template?
Currently, skilltemplates.com is a "passion project" - meaning; we don't make any money from it. We're happy to help as much as we can but we have day jobs and can't guarantee we'll always be availalble for support. So for now, support is limited to comments / questions on our YouTube channel which we normally respond to within 24-hours.
